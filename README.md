Pulseaudio for SqueezeOS
========================

I'd like to use Pulseaudio on my Logitech Squeezebox Radio (Baby).


Building Images
---------------

    $ git clone https://github.com/syssi/squeezeos.git
    $ cd squeezeos/poky
    $ source poky-init-build-env
    $ bitbake squeezeos-pulseaudio-image
